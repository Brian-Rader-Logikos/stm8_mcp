◆ BLDC_PWMDC_Set()

void BLDC_PWMDC_Set ( uint8_t  dc)

Sets motor speed from commanded throttle/UI setting.

The speed setting is checked for range and faults, if out of range, Stop() is called.

Parameters
dcSpeed input which can be in the range [0:255] but 255 is to be reserved for special use (out of band value). If the PWM resolution changes then the input speed will have to be re-scaled accordingly.
